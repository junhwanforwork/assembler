## 갭 목록

### G-1 · 경로 C 생성이 실사용 규모 아이디어에서 120s 타임아웃 — **HIGH · M1-C 편입 후보**
- **재현:** POST /api/generate — productId(코드-진실 API 25·테이블 7 참조) + 상세 아이디어(~600자, MAX_IDEA_LENGTH 이내) → 서버가 정확히 2.0min에 502 `ai_error`(1/1 재현).
- **원인 증거:** `src/lib/generate/run.ts:9` `GENERATE_TIMEOUT_MS = 120000` + opus·adaptive thinking·16K 출력 조합이 120s 초과. abort는 재시도 대상 아님(anthropic-retry.ts RETRYABLE_STATUS) → 단발 실패. 압축 아이디어도 103s — **정상 케이스가 한도의 86%**.
- **영향:** 신규 사용자의 첫 가치 경로(온보딩 C)가 내용이 충실할수록 실패. 2분 대기 후 실패라 체감 최악. rate limit 카운트도 소모됨.
- **후보 방향(진단만):** 스트리밍 전환(구 트랙 ASS-204 전례) 또는 타임아웃 상향+진행 피드백. FE 실패 카피 실측도 필요(미점검).

### G-2 · 아바타 이니셜 "J" 하드코딩 — **MED (거짓표면)**
- **재현:** 비로그인 세션으로 `/`·`/editor/[id]` — 우상단 아바타가 "J" 표시.
- **증거:** `src/components/editor/TopBar.tsx:67`·`src/components/dashboard/TopBar.tsx:14` `<Avatar initial="J" />`.
- **영향:** 익명 사용자에게 임의 정체성 표시 — 화면 언어=모델 언어 원칙(거짓 라벨 0) 위반. auth 배선 전엔 중성 표시(예: 비어 있는 아바타)가 정합.

### G-3 · 공유하기 disabled에 사유 없음 — **MED**
- **재현:** 에디터 TopBar "공유하기" — disabled(클릭 차단), title/aria-describedby 없음.
- **증거:** `src/components/editor/TopBar.tsx:64`. #63 확정은 "'곧' 비활성 노출"(방향 제시) — 현재는 이유 없는 죽은 버튼. ux-strategy 위임 기본값 "컨트롤급=disabled+**사유 툴팁**" 위반.

### G-4 · 코멘트 세그 disabled에 사유 없음 — **MED**
- **재현:** 인스펙터 "정보/코멘트" 세그의 코멘트 — disabled, 사유 없음. G-3과 동일 패턴(일괄 수정 가능).

### G-5 · 생성 실패의 FE 표면 미검증 — **MED (미점검 파생)**
- G-1이 API에서 확인된 만큼, Composer에서 2분 대기 중 진행 피드백·실패 카피·재시도 UX가 어떻게 보이는지 실측 필요(유료 호출이라 이번 점검에선 미실행). G-1 처리 티켓에 검증 시나리오로 포함 권장.

### G-6 · 추천 칩에 raw id 노출 — **LOW**
- **재현:** 도크 추천 칩 "req-e8963b72 수용 기준을 작성해요" — 모델 id가 사용자 문구에 그대로.
- **영향:** 화면 언어=모델 언어 원칙 위반(사용자는 id가 아니라 이름으로 인지). suggestions 프롬프트/렌더에서 제목 치환 필요.

### G-7 · API 카탈로그 행 인터랙션 부재 — **LOW**
- **재현:** 데이터>API 행 클릭 — 무반응(버튼 아님). 원본 #51 기대(행 아래 드릴인)는 프로토타입 배선 기준.
- **완화 요인:** "무엇을 하나요" 컬럼이 요약을 상시 노출해 드릴인 가치가 낮아짐. **editor-interactions #51 상태·기대 결과 재확정 후보**(문서 갱신은 오케스트레이터 몫).

### G-8 · 플로우 캔버스 우측 노드 잘림 의심 — **LOW (반쯤 미점검)**
- **재현:** 유저플로우 뷰 — 3번째 노드(대시보드)가 우패널 경계에 걸침. 가로 스크롤 제공 여부 미확인. 노드가 더 많은 그래프에서 재확인 필요.
